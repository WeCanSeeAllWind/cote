# Coding Test

### 기술 스택

| 파트       | 내용              |
| ---------- | ------------------|
| 프론트엔드 | `Nginx`              |
| 백엔드    | `PHP-FPM`,`Docker`   |
| DB      | `MySQL`              |


### 실행방법

#### docker-compose 실행

`docker-compose up`

#### 초기 DB 생성: 브라우저에서 아래 URL로 접속

`http;//localhost/seeder.php`

#### 오래된 데이터 옮기기: 브라우저에서 아래 URL로 접속

`http;//localhost/transfer.php`
